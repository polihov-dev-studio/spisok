# Private Buylist

Готовый статический сайт для GitHub Pages.

## Что умеет
- вход по паролю `1234`
- категории и подкатегории
- ручное добавление товаров
- ссылка, фото, цена, количество, заметка
- фильтры
- избранное
- суммы по подкатегории, категории и по всему списку
- данные сохраняются в `localStorage`

## Как загрузить на GitHub Pages
1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория файлы:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Открой **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / root
5. Сохрани.
6. Через минуту появится ссылка на сайт.

## Важно
Это не настоящая безопасная авторизация. Пароль хранится в коде и нужен только как лёгкая защита от посторонних глаз.

## Как поменять пароль
Открой `script.js` и `index.html`.

В `script.js` измени строку:
```js
const APP_PASSWORD = '1234';
```

В `index.html` при желании измени текст-подсказку с паролем на экране входа.
